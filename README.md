## 1. Project Overview
**Project Name:** SportsIn Portfolio Website  
**Type:** Graduation Project CMS Website (HubSpot CMS)

This project is an introductory **portfolio website** for the graduation project **SportsIn**. The website content presents the graduation project idea, MVP, technology stack, and team members.

The website was developed using **HubSpot CMS**, focusing on dynamic content, scalability, and easy content management for non-technical users. The goal is to provide a professional digital presence for the graduation project and demonstrate CMS and web development skills.

---

## 2. Problem Statement
Graduation projects often lack a professional digital portfolio to present their work effectively. Static websites are inflexible and do not allow easy updates by content editors.

The challenge was to build a dynamic, editable website that:
- Presents the SportsIn graduation project clearly
- Enables content updates without coding
- Follows HubSpot CMS best practices

## 3. Project Objectives

- Use HubSpot CMS Design Manager to create reusable drag-and-drop modules
